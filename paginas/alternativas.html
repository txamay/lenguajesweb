HTML presenta numerosas posibilidad que se pueden utilizar como alternativa a Flash. Entre ellas destacan:
Canvas
Es un elemento HTML que puede ser usado para dibujar gráficos vectoriales, incluyendo animaciones, usando scripts (normalmente JavaScript)
WebGL
Es una especificación estándar que define una API para la renderización de gráficos en 3D dentro de cualquier navegador web
SVG
Es un formato de gráficos vectoriales bidimensionales, tanto estáticos como animados, que almacena sus datos en formato XML
3D CSS Transforms
Permiten transformar (desplazar, escalar, girar...) objetos sobre un sistema de coordenadas tridimensional
SMIL
Permite realizar animación, inserción de objetos multimedia, temporización y sincronización, y efectos de transición
Audio y vídeo nativos
HTML5 introduce soporte integrado para el contenido multimedia gracias a los elementos <audio> y <vídeo>
