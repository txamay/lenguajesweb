Antecedentes

El precursor de Flash es un software de dibujo vectorial publicado por FutureWave Software , que se presentó con dos nombres:

SmartSketch
  • Versión para el SOPenPoint
  • Versión para los SO Windows y MacOS
FutureSplash

Macromedia

En noviembre de 1996 FutureSplash es adquirida por Macromedia , que presenta Flash 1.0. Esta herramienta está compuesta por:

Macromedia Flash
Un editor de gráficos, que permite realizar animaciones con ellos
Macromedia Flash Player
Un visualizador para dichos gráficos
Flash 5. Presenta la primera versión del lenguaje ActionScript
Flash MX 2004 Presenta ActionScript 2.0 que incluye:
  • Programación orientada a objetos
  • Componentes para una UI mejorada
  • Toda una serie de características avanzadas/li>
Flash 8. Última versión de Macromedia

Adobe

Dreamweaver
Director

Adobe Flash CS3 Professional que presenta a ActionScript 3.0
